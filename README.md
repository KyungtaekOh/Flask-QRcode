# Flask-QRcode
##Flask app for creating and recognizing QR codes

### Install Package
```
pip install flask
pip install sqlalchemy
pip install opencv-python
pip install pyzbar
```


### Install sqlite3
```
python
from fapp import fdatabase
fdatabase.create_all()
```

### Run
```
flask run
(if need setting)
set FLASK_APP=fapp
```
